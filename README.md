# Microsoft Graph API PoC

## Technology

- Microsoft Graph API,
- Node.js,
- TypeScript,
- Express server,
- Localtunnel for creating HTTPS endpoint

## Features

- Microsoft Graph OAuth 2.0 flow authentication,
- Storing user access tokens
- Sending custom Outlook emails as a User
- Accessing Outlook Calendar
